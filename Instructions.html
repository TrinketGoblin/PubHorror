PUBHORROR

1. Initial Setup

  • Launch the application and locate the Settings (⚙️) icon.
  • Click Set Library Path and select the folder where your movies are stored.
  • Paste your TMDB API Key in settings to enable automatic trailer fetching.

2. Managing Files

  • Use the Plus (+) button to add individual files or entire folders.
  • Click Import Folder to bulk-scan your library for media.
  • The app supports formats like .mp4, .mkv, and even .ac3 audio files.

3. Customization

  • Adjust the Font Size Slider for better readability.
  • Choose from Horror, Midnight, or Forest themes in the options bar.
  • Right-click movies to edit notes or export jump-scare timestamps.
