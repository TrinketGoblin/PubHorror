PUBHORROR

Streamlined management for your nightmare collection.

1 Set Your Library Path

Before PubHorror can find your films, you must tell it where they live. By default, the app looks in C:\Users\Public\Programs\library.

  • Open the Settings (⚙️) in the top right.
  • Click Set Library Path and select the folder containing your .mp4 or .mkv files.

2 Adding Content

Use the Plus Button (＋) to populate your library:

  • Add Movie: Select a single video file.
  • Add Series: Select a folder containing TV episodes.
  • Import Folder: Bulk import an entire directory of horror films.

3 Personalize the Void

The interface is fully adjustable to suit your preferences:

  • Themes: Switch between Midnight (Blue), Forest (Green), or Classic (Grey) via the Options panel.
  • Accessibility: Use the Font Size Slider in the options bar to adjust text for better readability.
