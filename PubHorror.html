🎞 PUBHORROR

The ultimate personal horror movie library manager.

Welcome to the Void

PubHorror is a specialized media manager for horror enthusiasts. It scans your local files, organizes them into collections, and lets you track your "watched" status with a spooky, customizable interface.

Getting Started

  1. Download: Clone the repository or download the latest executable.
  2. Set Your Library: Open Settings (⚙️) and point the library path to your movie folder.
  3. Add Content: Use the ＋ button to add individual movies, entire series folders, or import existing JSON libraries.

Key Features

  • Thematic UI: Switch between Horror, Midnight, and Forest themes.
  • Library Stats: View your viewing habits and collection totals.
  • Trailers: Fetch trailers directly via TMDB integration.
  • Jump Scare Tracking: (Internal Feature) Keep track of the scares in your collection.
