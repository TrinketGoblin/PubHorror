PUBHORROR

The accessible horror movie manager

1. Connection

  • Launch the program and click Settings (⚙️).
  • Select Set Library Path to link your movie folder.
  • Add your TMDB API Key to enable automatic trailer fetching.

2. Import

  • Use the Plus (+) icon to add files.
  • Choose Import Folder for bulk scanning of your directory.
  • Right-click any poster to Refresh Metadata if the cover art is missing.

3. Viewing

  • Click a poster to see the Scare Details and notes.
  • Hit Play to open the movie in your default player.
  • Toggle Fullscreen in the options for a cinema experience.
